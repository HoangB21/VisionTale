为文本生成配图场景片段，返回JSON格式：

1. 描述词规则：
- 人物动作优先，人名用{}包裹
- 仅使用画面感强的动词
- 仅保留关键视觉特征的形容词
- 不包含心理描写和情绪
- 从已有场景列表选择一个基底场景，用$$包裹
- 可添加场景细节（如夜色、月光）
- 用逗号分隔元素
- 不包含对话

2.注意事项
- 尝试根据上下文或者已有实体列表识别每段文本中的人称代词，生成提示词的时候将人称代词替换为具体名字
- 理论上，如果不是特殊情况，那么每段文本地基底场景应该是一样的，除非文本有明显的场景转换意图

[已有场景列表]
{scenes}

[已有实体列表]
{entities}

输出格式：
{
  "spans": [
    {
      "content": "文本内容",
      "base_scene":"基底场景",//只需要一个
      "scene": "描述词（包含场景细节、人物、动作）"//注意,scene不包含任何基底场景
    }
  ]
}

示例：
输入：林小夏脸一红，低声嘟囔：“谁在看他啊……"
      苏晴笑着戳了戳她的肩膀：“别装了，你的眼神早就出卖你了。"
输出：
{
  "spans": [
    {
      "content": "林小夏脸一红，低声嘟囔：“谁在看他啊……",
      "base_scene":教学楼台阶
      "scene": "{林小夏}脸红, 低声嘟囔"
    },
    {
      "content":"苏晴笑着戳了戳她的肩膀：“别装了，你的眼神早就出卖你了。"
      "base_scene":教学楼台阶
      "scene": "{苏晴}微笑地戳{林小夏}肩膀"
    }
  ]
}
